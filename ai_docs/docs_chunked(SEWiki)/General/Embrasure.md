The Embrasure block is a decorative block in Space Engineers. It is only available if you purchase the [Warfare 1 Pack](https://spaceengineers.wiki.gg/wiki/Warfare_1_Pack "Warfare 1 Pack") DLC.

## Usage

The Embrasure serves as a window for players to fire through while still maximizing cover from incoming fire.

## Construction

The Embrasure exists for large grid only. This block needs 1x1x1 blocks room to be placed but has the volume of an upright half-block.
