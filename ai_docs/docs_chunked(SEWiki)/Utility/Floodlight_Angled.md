The Floodlight Angled is a light source block that is part of the [Floodlights](https://spaceengineers.wiki.gg/wiki/Floodlights "Floodlights") set. You can build these blocks only if you purchase the [Contact Pack](https://spaceengineers.wiki.gg/wiki/Contact_Pack "Contact Pack") DLC.

## Usage

For large grid, the minimum light beam is 10 metres and the maximum is 160 metres. For small grid, the minimum light beam is 10 metres and the maximum is 120 metres. The default is 120 metres. For usage details, see [Spotlight](https://spaceengineers.wiki.gg/wiki/Spotlight "Spotlight").

## Construction

The Floodlight Angled faces 45 degrees down.

These light sources exist for large and small grid. On large grid, they look like sets of four lights and on small grid it is one. The large grid blocks have mount points at the back and both sides. The small grid blocks only have mount points on the backside.
